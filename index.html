Frequently Asked Questions

Here you will get to know about some front end related question answers

  • Flex box is used for one dimentional layout. On the other hand, grid is used for two dimentional layout
  • Flex box is supported by the most of the browsers whereas grid is not supported by the old browser versions
  • Flex box is mostly used for aligning contents whereas grid is used for 2d layouts

  • Bootstrap comes with predesigned components whereas Tailwind comes with pre-styled widgets
  • Websites designed with Bootstrap will look identical whereas websites designed with Tailwindwill look different
  • Bootstrap is older. Tailwind is relatively newer
  • Bootstrap has lesser options for spacing and coloring compared to Tailwind

The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.
  • Content - The content of the box, where text and images appear
  • Padding - Clears an area around the content. The padding is transparent
  • Border - A border that goes around the padding and content
  • Margin - Clears an area outside the border. The margin is transparent
